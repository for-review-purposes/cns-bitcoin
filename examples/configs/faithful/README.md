# Faithful Bitcoin Configuration as of `2024-11-25` for Simulator Implementation Validation via Block Pace Comparison with Empirical



This configuration simulates the Bitcoin network as of a specific date, via setting simulation parameters according to network measures that can be retrieved by online statistics aggregators. Parameters that are faithfully coped - modulo distributional and dispersion assumptions - include difficulty, total network power, and transaction rate, size, and fee.



The analysis report can be read in [BitcoinValidation.md](https://github.com/for-review-purposes/cns-bitcoin/blob/main/examples/configs/faithful/BitcoinValidation.md). The report can be reproduced by `knitting` (i.e., running) R markdown script `BitcoinValidation.Rmd` that exists in the same directory.
