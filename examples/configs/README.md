## Experimental Markdowns and Data for CCS 2026 submission #1065: *"Finality-oriented simulation-based evaluation of blockchain consensus networks: the CNS framework"*



This folder contains reproducibility scripts and data for the three experiments presented in the paper. The markdowns, which contain the analysis steps, results, and reproducibility directions are as follows:

* Validation Experiment: Bitcoin's Block Pace can be found under [faithful/BitcoinValidation.md](https://github.com/for-review-purposes/cns-bitcoin/blob/main/examples/configs/faithful/BitcoinValidation.md).
* Experiment 1: Block Size vs. Finality can be found under [size/blockSize.md](https://github.com/for-review-purposes/cns-bitcoin/blob/main/examples/configs/size/blockSize.md).
* Experiment 2: Censorship Attack can be found under [attack/HiddenChainAttack.md](https://github.com/for-review-purposes/cns-bitcoin/blob/main/examples/configs/attack/HiddenChainAttack.md).

Each of the above markdown files are a product of an R or Quarto markdown in the same directory.
